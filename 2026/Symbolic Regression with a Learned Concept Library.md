该论文提出 **LASR**（Learned Abstraction for Symbolic Regression），一种用于**符号回归**（Symbolic Regression, SR）的新方法。其核心思想是**通过大语言模型**（LLM）来**发现和演化一个抽象的、可复用的自然语言概念库**，并利用该概念库来指导和加速符号回归的搜索过程。

LASR 的工作流程是一个**交替优化的循环**，包含三个阶段。首先，在**假设演化**（Hypothesis Evolution）阶段，系统以标准的遗传算法（如 PySR）为基础，但在其随机的变异和交叉操作中，以一定概率**注入 LLM 引导的操作**。这些 LLM 操作会参考当前概念库中的概念，生成更符合科学直觉的数学表达式。其次，在**概念抽象**（Concept Abstraction）阶段，系统分析当前表现最好和最差的数学表达式，并通过**零样本**（zero-shot）的 LLM 查询，从中**抽象出高层次的、可解释的自然语言概念**（例如，“该表达式表现出与温度相关的指数衰减”），并将其加入概念库。最后，在**概念演化**（Concept Evolution）阶段，系统再次利用 LLM，对概念库中已有的概念进行交叉和演化，生成新的、更普适或更简洁的概念，以进一步丰富和优化概念库。

该方法有效解决了传统符号回归方法的两大局限。首先，它克服了**纯随机搜索**的低效性，通过 LLM 引入的科学先验知识，将搜索引导至更有可能包含正确解的区域。其次，它避免了**依赖预定义的、静态的**问题设定，而是通过一个**动态的、可学习的**概念库，使系统能够从过往的搜索经验中学习并不断改进其搜索策略。

作者在多个基准上对 LASR 进行了评估。在**费曼方程**（Feynman Equations）这一经典符号回归基准上，LASR 成功发现了 66 个方程，超越了现有最佳方法（59 个）。在**合成数据集**上，为了排除数据泄露的可能，LASR 依然显著优于基线方法，证明了其泛化能力。此外，作者还进行了一项**案例研究**，使用 LASR 在 BIG-Bench 数据集上**发现了一条新的 LLM 缩放律**，揭示了模型训练步数与推理时示例数量（shots）之间的新关系，展示了其在科学发现中的实际应用潜力。

与依赖固定启发式或纯数据驱动的方案不同，LASR 将 LLM 的**知识引导**与**进化算法的鲁棒性**相结合，提供了一种高效、可解释且能持续学习的符号回归新范式。项目代码已开源。
